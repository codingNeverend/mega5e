Le système du jeu de rôle MEGA !!!

## Introduction

Bienvenue dans le système de jeu de rôle **MEGA** pour **FoundryVTT** !

## Fonctionnalités

- **Création de personnage** : Créez et personnalisez vos personnages avec des options variées.
- **Gestion des combats** : Gérez les combats de manière fluide et intuitive.
- **Magie et compétences** : Utilisez des sorts et des compétences uniques pour chaque personnage.
- **Intégration FoundryVTT** : Profitez d'une intégration complète avec FoundryVTT pour une expérience de jeu optimale.

## Installation

1. Téléchargez le système depuis [le site officiel](https://example.com/download).
2. Ouvrez FoundryVTT et allez dans l'onglet **Systèmes de Jeu**.
3. Cliquez sur **Installer un système** et sélectionnez le fichier téléchargé.
4. Créez un nouveau monde en sélectionnant le système **MEGA**.

**Note importante :** À partir de cette version, le module MegaSocket est intégré directement dans le système. Vous n'avez plus besoin d'installer de module externe séparé.

## Utilisation

1. Créez un nouveau monde ou ouvrez un monde existant dans FoundryVTT.
2. Allez dans les **Paramètres du monde** et activez le module **MEGA System**.
3. Commencez à créer des personnages et à jouer !
